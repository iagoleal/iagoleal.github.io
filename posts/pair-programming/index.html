My Pair Programming Setup for the Pandemic

15 July 2021

Tmux Pair Programming

My pair programming setup during the pandemic.

Done on Arch Linux

ssh + tmux + nvim

Steps on the remote side

Ensure everything is installed

# pacman -S openssh tmux

Also, ensure that you have a terminal based text editor: neovim, vi, vim, emacs, nano, ed etc.

Setup an user for the remote client

Here we will call him outsider, but you may choose any name you want.

# adduser outsider # passwd outsider

Setup a group with both your user and the new user

# groupadd tmux # usermod -a -G tmux youruser # usermod -a -G tmux outsider

ngrok

Download the ngrok binary, create a ngrok account and follow the tutorial on their site to setup everything.

Start the server side (on your machine)

First start the ssh daemon (This is systemd specific)

$ systemctl start sshd

Start ngrok tunnel

$ ngrok tcp 22

Copy the port and host names from the window.

Start tmux with a shared session

$ tmux -S /tmp/sharedtmux new-session -s “$tmux_session_name” -d

Let the tmux group own this file

$ chgrp tmux_grp "tmux_session_file" $ chmod g+rw “$tmux_session_file”

Attach to it

$ tmux -u -S $tmux_session_file attach -t $tmux_session_name

Client side

Al the client has to do is run the ssh command plus the tmux attach command.

$ echo ssh -t -p “port" "remote_user”@“$host” “‘tmux -u -S $tmux_session_file attach -t $tmux_session_name’”
